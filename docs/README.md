## 🚀 Quick Start

### For New Developers
1. Start with the [Project Overview](./architecture/PROJECT_OVERVIEW.md) to understand the architecture
2. Review [Contributing Guidelines](./project/CONTRIBUTING.md) for development standards
3. Check [Deployment Checklist](./deployment/DEPLOYMENT_CHECKLIST.md) for environment setup

### For Deployment
1. Review [Production Deployment Checklist](./deployment/PRODUCTION_DEPLOYMENT_CHECKLIST.md)
2. Follow [Email Setup Guide](./guides/EMAIL_SETUP_GUIDE.md) for email configuration
3. Configure [S3 Setup](./guides/S3_SETUP_GUIDE.md) if using file storage

### For Troubleshooting
1. Check [Fixes](./fixes/) directory for known issues and solutions
2. Review [Development](./development/) guides for debugging workflows
3. See [Deployment](./deployment/) for deployment-specific issues
